# Маргулан — ценности и принципы (reference)

Этот документ вынесен из USER.md, чтобы сохранить смысл, но не грузить длинный текст в каждый запрос.

## Иерархия (на всю жизнь)

1) Цель жизни: стать автором концепции «Суперпозиция».
2) Миссия: исследовать законы мира, выявлять эффективное и делиться этим.
3) Стандарт жизни: возможностей больше, чем потребностей.
4) Финансовые метрики стандарта:
   - > $200 млн net worth
   - > $1 млн/мес доход

## Ценности в людях

1) Порядочность (честность, ответственность, скромность, преданность)
2) Ум (здравый смысл, любопытство, изобретательность, саморазвитие)
3) Энергия (оптимизм, эмоциональная доброта, бодрость, терпение)

## Принципы инвестирования (кратко)

- «Если не точно Да — значит Нет»
- Долгосрок > краткосрок
- Тайминг важнее цены; тренд важнее актива
- Покупать качество, не цену
- Диверсификация; всегда иметь опцион
- Ставка на наездника, не на лошадь
- Как можно дольше оставаться в игре; не рисковать всей игрой
- Нельзя делать сделку с плохим человеком

## Стиль управления/работы (кратко)

- Управление: убеждение; вмешательство по отклонениям
- Работа: стратегический, воспитательный стиль; задавать цели/срок/цену, остальное — ресурс/поддержка

(Полные формулировки можно расширить по запросу.)
