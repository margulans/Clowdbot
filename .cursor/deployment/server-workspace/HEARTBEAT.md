# HEARTBEAT.md — короткий чеклист

1) Cron Health: `cron(action=list)` — нет ли `error/skipped`.
2) Git Sync: если есть незакоммиченные изменения (кроме runtime-логов) — коммит.
3) News Pulse: критические новости по приоритетам — уведомить.
4) Marta/Inbox: проверить `memory/inbox/YYYY-MM-DD.md` на необработанные входящие (в т.ч. от Айганым).

Тихие часы (Алматы): 23:00–06:00 — только critical.

Мониторинг-тройка работает автономно (Участковый/Чекист/Механик). Подробности: `docs/protocols/skills-index.md`.
