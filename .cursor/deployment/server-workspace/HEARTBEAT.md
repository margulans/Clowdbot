# HEARTBEAT.md

## Системный чеклист
1. **Cron Health:** Проверь `cron(action=list)`, нет ли пропусков (lastRunAtMs).
2. **Git Sync:** Если есть незакоммиченные файлы — `git commit`.
3. **News Pulse:** Если не время дайджеста, но есть критические новости по ИИ/eVTOL — уведомь.

## ⚕️ Авиценна (Раз в 7 дней)
- Проверь `skills/avicenna/data/schedule.md` на предмет чекапов (≤14 дней).
- Проверь `medications.md` — не заканчивается ли курс (≤7 дней).
- Раз в 30 дней: проверь дату последних анализов в `checkups.md`.

## Правила тишины
- **23:00 - 08:00 (Алматы):** Только критические алерты.
- **Во время дайджестов:** Пропускай рутинные проверки heartbeat.

Детали мониторинга источников — в `docs/protocols/digest-operations.md`.
